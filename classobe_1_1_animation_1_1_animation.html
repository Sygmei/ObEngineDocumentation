Parameters
parametersA vili::node that contains the following optional parameters :
  • spriteOffsetX : x Coordinate of the Sprite Offset in the Animation in pixels.
  • spriteOffsetY : y Coordinate of the Sprite Offset in the Animation in pixels.
  • priority : Priority of the Animation (A higher Animation priority can't be interrupted by an Animation with a lower one).
