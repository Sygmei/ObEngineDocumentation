◆ Units

enum obe::Transform::Units
strong

Different Units used to work with elements position / size in the Scene.

Enumerator
ViewPercentage 

ViewPercentage is an Unit used to place / scale an element using the View.
Exemple : x = 0.5, y = 0.5 is the middle of the View.

ViewPixels 

ViewPercentage is an Unit used to place / scale an element using the pixels of the View.
Exemple : x = 500, y = 500 is the middle of a 1000x1000 pixels screen.

ViewUnits 

ViewUnits is an Unit used to place / scale an element using the View.
Exemple : x = 1, y = 1 is the middle of a 1000x1000 pixels screen.
The screen height will ALWAYS be 2 ViewUnits when the View size is 1, the width will depend on the screen ratio.
The screen width in ViewUnits can be calculated using this formula : 2 * viewSize * (Screen.Width / Screen.Height)

WorldPixels 

ViewPercentage is an Unit used to place / scale an element using the pixels of the Scene.
Exemple : x = 4000, y = 10000.

WorldUnits 

WorldUnits is an Unit used to place / scale an element using the Scene.
Exemple : x = 4.5, y = 2.5.

Unknown 

This Unit should never be used. (Debug purpose)
