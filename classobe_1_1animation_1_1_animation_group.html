◆ next()

void obe::animation::AnimationGroup::next ( bool  force = false)

Increment index of the current texture to be displayed.

  • If max index is reached and no more loops, the AnimationGroup will be over.
  • If max index is reached but there is some loops left, index will go back to 0.
    Parameters
    forceForce increments the index of the AnimationGroup without waiting for the delay if force is true

◆ previous()

void obe::animation::AnimationGroup::previous ( bool  force = false)

Decrement index of the current texture to be displayed.

  • If index reach 0 and no more loops, the AnimationGroup index will stay at 0.
  • If index reach 0 and there is some loops left, index will go at get_textures_amount() - 1.
    Parameters
    forceForce decrements the index of the AnimationGroup without waiting for the delay if force is true
